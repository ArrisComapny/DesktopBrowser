# 🛒 Настройка Личных Кабинетов для DesktopBrowser

Данное руководство поможет подготовить кабинеты маркетплейсов (Wildberries, Ozon, Yandex.Market) для работы с приложением **DesktopBrowser**.

---

## 🔷 Wildberries (WB)

### Что требуется:
- Зарегистрировать аккаунт на https://www.wildberries.ru
- Добавить этот аккаунт как пользователя в ЛК:  
https://www.seller.wildberries.ru/supplier-settings/supplier-users

> 💡 Рекомендуется аккаунт использовать **только для одного кабинета**.  
> Для одного кабинета можно создавать **несколько аккаунтов**.
> 
### Подготовка:
1. Убедитесь, что номер телефона **актуален** и доступен для получения SMS
2. Добавьте или обновите данные в таблице `connects`:
   - `phone` — номер без +7
   - `proxy` — уникальный для каждого кабинета
   > Запись в таблице `connects` может быть использована для подключения одного кабинета на разных платформах

3. В таблицу `markets`:
   - `marketplace` = `WB`
   - `name_company` = Название кабинета (для каждого кабинета + аккаунта задайте своё название)
      > Пример: MyCompanyWarehouse; MyCompanyFinance; MyCompany
   - `phone` = номер аккаунта из `connects`

## 🔷 Ozon

### Что требуется:
- Зарегистрировать и настроить email на Yandex 📮 [Инструкция по настройке Yandex-почты](setup_yandex_mail.md)
- Зарегистрировать аккаунт на  https://www.ozon.ru
- Добавить Yandex-почту в настройках аккаунта Ozon:  
https://ozon.ru/my/settings и https://ozon.ru/ozonid
- Добавить этот аккаунт как пользователя в ЛК:  
https://seller.ozon.ru/app/settings/employees

> Аккаунт Yandex-почты и Ozon рекомендуем регистрировать на один номер

### Подготовка:
1. Убедитесь, что номер телефона **актуален** и доступен для получения SMS
2. Убедитесь, что Yandex-почта **актуальна** и доступна для получения кода
3. Добавьте или обновите данные в таблице `connects`:
   - `phone` — номер без +7
   - `proxy` — уникальный для каждого кабинета
   - `mail` — Yandex-почта для получения кодов на Ozon
   - `token` — токен доступа к Yandex-почте, для получения кодов на Ozon
   > Запись в таблице `connects` может быть использована для подключения одного кабинета на разных платформах

4. В таблицу `markets`:
   - `marketplace` = `Ozon`
   - `name_company` = Название кабинета (для каждого кабинета + аккаунта задайте своё название)
      > Пример: MyCompanyWarehouse; MyCompanyFinance; MyCompany
   - `phone` = номер аккаунта из `connects`

## 🔷 Yandex.Маркет (Yandex)

### Что требуется:
- Зарегистрировать и настроить email на Yandex 📮 [Инструкция по настройке Yandex-почты](setup_yandex_mail.md)
- Добавить этот аккаунт как пользователя в ЛК Yandex.Market

### Подготовка:
1. Убедитесь, что номер телефона **актуален** и доступен для получения SMS
2. Добавьте или обновите данные в таблице `connects`:
   - `phone` — номер без +7
   - `proxy` — уникальный для каждого кабинета
   - `mail` — Yandex-почта для входа в ЛК
   - `pass_mail` — пароль Yandex-почты для входа в ЛК
   > Запись в таблице `connects` может быть использована для подключения одного кабинета на разных платформах

3. В таблицу `markets`:
   - `marketplace` = `Yandex`
   - `name_company` = Название кабинета (для каждого кабинета + аккаунта задайте своё название)
      > Пример: MyCompanyWarehouse; MyCompanyFinance; MyCompany
   - `phone` = номер аккаунта из `connects`

---

## 💡 Примечания

- Для каждого кабинета рекомендуется использовать **отдельный proxy**
- Все значения `name_company` должны быть **уникальны в рамках одного маркетплейса**
- Телефон и Email должны быть активны, иначе авторизация может завершиться сбоем
