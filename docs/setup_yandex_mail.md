# 📮 Настройка Yandex-почты для DesktopBrowser

Этот документ описывает, как создать Yandex-почту и получить пароль приложения для доступа по IMAP/SMTP. Он необходим для автоматического получения кодов авторизации (например, от Ozon).

---

## 📧 1. Создание Yandex-почты

1. Перейдите на [https://passport.yandex.ru/registration](https://passport.yandex.ru/registration)
2. Заполните данные: имя, логин, пароль, номер телефона
3. Подтвердите регистрацию

> 💡 Рекомендуется создавать отдельную почту под каждую компанию

---

## 🔐 2. Генерация пароля приложения

Для доступа по IMAP/SMTP требуется отдельный безопасный пароль.

1. Перейдите в [настройки безопасности аккаунта](https://passport.yandex.ru/profile)
2. Выберите вкладку **"Пароли приложений"**
3. Нажмите **"Создать пароль"**
4. Выберите тип доступа: **Почта — IMAP, POP3, SMTP**
5. Назовите пароль (например: `API`)
6. Скопируйте сгенерированный пароль

> ❗️ Этот пароль отображается **только один раз**. Обязательно сохраните его сразу.

---

## ✉️ 3. Включение доступа по IMAP

1. Зайдите в [настройки почты Yandex](https://mail.yandex.ru)
2. В разделе **Почтовые программы** включите:
   - ✅ С сервера imap.yandex.ru по протоколу IMAP
   - ✅ Пароли приложений и OAuth-токены

---

## 📌 Использование в приложении

Добавьте эти данные в таблицу `connects`:

| Поле        | Значение                                         |
|-------------|--------------------------------------------------|
| `mail`      | Ваша Yandex-почта                                |
| `token`     | Пароль приложения (IMAP/SMTP)                    |
| `pass_mail` | Пароль от почты (исп. только для Яндекс.Маркет)  |

---
